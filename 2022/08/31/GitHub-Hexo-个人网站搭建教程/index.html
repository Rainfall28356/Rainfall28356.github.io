GitHub+Hexo 个人网站搭建教程

2022-08-31
学习历程 / 前端
Word count: 1.5k   |   Reading time≈ 5 min

网络时代，拥有一个属于自己的网页，并用以自由发挥、记录生活，一定是许多程序猿的美好愿景。本人也在大二暑假开始尝试实现这一愿景，并已经初见成效。

现分享个人搭建网站流程：

# 1 GitHub 创建仓库

首先登录 GitHub, 在个人主页下点击 Your repositories 后点击 New 新建工程。

进入到设置页面，只需要填写仓库名为： <你的用户名>.github.io 即可，其他全部默认就好。

# 2 Git 安装

Git 是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。我们需要通过 Git 将本地的网站同步到 GitHub 上。
直接选择从 Git 官网下载。

下载完成后可在 Windows 控制台输入’git --version’检查是否安装成功。

Node.js 下载地址，可同时安装 Node.js 和 npm (软件包管理器)。
在 windows 控制台输入 node -v 和 npm -v 可分别检验二者是否成功安装。

倘若出现 npm 报错情况，可参照这篇博客解决。

# 6 优化支持

至此，一切都已就绪，你可以开始好好打磨自己的小空间了！这里为大家提供一些个人的优化建议。

主题

Hexo 提供了很多主题 Themes，每个主题的制作者都在其博客提供了非常详细的布置教学，可以依个人喜好选择性的学习。

Markdown 语法

Hexo 使用 Markdown 语法，这是一种十分方便的标记语言，还可以与 HTML 混编。
参考 Markdown 官网学习，还有在线编辑器可以使用预览，很快就能掌握最常用的几个语法。

图床

一般来说，在 Hexo 插入大量图片可以选择外链图床的方法，这样可以优化网页的加载时间。
聚合图床是本人正在使用的一个图床网站，可以直接截图粘贴生成 Markdown 语法链接，非常方便。

# 7 新的开始

舞台搭建完毕，终于可以开始写博客啦！
写博客贵在坚持，网上的许多博主只坚持了一两年就停止了更新，非常可惜。
本人认为，写博客能够梳理自己的思路，理清学习到的知识，更能指导、帮助其他人，很有积极意义。
很多事物最终都将随着时间消散，互联网也能成为我们缅怀自己过去的载体。

所以，心动不如行动，快来着手搭建起自己的网站吧！
