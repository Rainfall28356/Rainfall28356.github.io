# 1 三角函数的正交性

在解析几何中，定义两个向量正交可用点积（内积）来表示：$$\vec a \cdot \vec b = 0$$

将正交的概念推广到连续函数上，在区间x∈\[a,b\]，如果函数 f (x) 和 g (x) 正交，则 $$\int_a^bf (x) g (x) dx = 0$$

此时定义三角函数系（省略 0）：$${1 (cos0x),sinx,cosx,sin2x,cos2x,…,sinnx,cosnx,…}$$

可以知道任何两个不同函数在x∈\[-π,π\] 内正交，即：$$\int_-π}【πsinnxcosnxdx = 0$$ $$\int_{-π】^πcosnxcosmxdx = 0(n\not=m)$$ $$\int_-π}【πsinnxdx = \int_{-π】^π1×sinnxdx = 0$$

注意只有当两个三角函数不相同时才可在x∈\[-π,π\] 内正交，当 n = m 时（积化和差）：

∫−ππcosnxcosnxdx=∫−ππ12(1+cos2nx)dx=12(∫−ππ1dx+∫−ππcos2nxdx)=12∫−ππ1dx=π\begin{aligned} \int_{-π}^πcosnxcosnxdx &= \int_{-π}^π{1 \over 2}(1+cos2nx)dx \\ &= {1 \over 2}(\int_{-π}^π1dx + \int_{-π}^πcos2nxdx) \\ &= {1 \over 2}\int_{-π}^π1dx \\ &= π \end{aligned}∫−ππ​cosnxcosnxdx​=∫−ππ​21​(1+cos2nx)dx=21​(∫−ππ​1dx+∫−ππ​cos2nxdx)=21​∫−ππ​1dx=π​

# 2 周期为 2π 的函数展开

对于一个周期 T = 2π 的函数：$$f_T (x) = f_T (x + 2π)$$

它总能用三角级数表达：$$f_T (x) = \sum_n = 0}【\infty a_ncosnx + \sum_{n = 0】^\infty b_nsinnx$$

首先求解特殊项 a_0^*。当 n = 0 时：

fT(x)=a0∗cos0+b0sin0+∑n=1∞(ancosnx+bnsinnx)=a0∗+∑n=1∞(ancosnx+bnsinnx)\begin{aligned} f_T(x) &= a_0^*cos0 + b_0sin0 + \sum_{n = 1}^\infty (a_ncosnx + b_nsinnx) \\ &= a_0^* + \sum_{n = 1}^\infty (a_ncosnx + b_nsinnx) \end{aligned}fT​(x)​=a0∗​cos0+b0​sin0+n=1∑∞​(an​cosnx+bn​sinnx)=a0∗​+n=1∑∞​(an​cosnx+bn​sinnx)​

对等式两边从 -π 到 π 作定积分，得到：

∫−ππfT(x)dx=∫−ππa0∗dx+∫−ππ∑n=1∞(ancosnx+bnsinnx)dx=∫−ππa0∗dx+∑n=1∞an∫−ππcosnxdx+∑n=1∞bn∫−ππsinnxdx\begin{aligned} \int_{-π}^πf_T(x)dx &= \int_{-π}^πa_0^*dx + \int_{-π}^π\sum_{n = 1}^\infty (a_ncosnx + b_nsinnx)dx \\ &= \int_{-π}^πa_0^*dx + \sum_{n = 1}^\infty a_n \int_{-π}^πcosnxdx + \sum_{n = 1}^\infty b_n \int_{-π}^πsinnxdx \end{aligned}∫−ππ​fT​(x)dx​=∫−ππ​a0∗​dx+∫−ππ​n=1∑∞​(an​cosnx+bn​sinnx)dx=∫−ππ​a0∗​dx+n=1∑∞​an​∫−ππ​cosnxdx+n=1∑∞​bn​∫−ππ​sinnxdx​

根据三角函数的正交性，可以知道∑n=1∞an∫−ππcosnxdx=∑n=1∞bn∫−ππsinnxdx=0\sum_{n = 1}^\infty a_n \int_{-π}^πcosnxdx = \sum_{n = 1}^\infty b_n \int_{-π}^πsinnxdx = 0∑n=1∞​an​∫−ππ​cosnxdx=∑n=1∞​bn​∫−ππ​sinnxdx=0，因此：$$\int_-π}【πf_T(x)dx = \int_{-π】^πa_0dx$$ $$a_0^* = {1 \over 2π} \int_{-π}^πf_T(x)dx$$

下面求解通项 a_n。对等式两边同时乘以 cosmx 后再从 -π 到 π 作定积分，得到：$$\int_-π}【πf_T(x)cosmxdx = a_0^*\int_{-π】^πcosmxdx + \sum_n = 1}【\infty a_n \int_{-π】^πcosnxcosmxdx + \sum_n = 1}【\infty b_n \int_{-π】^πsinnxcosmxdx$$

对于∑n=1∞an∫−ππcosnxcosmxdx\sum_{n = 1}^\infty a_n \int_{-π}^πcosnxcosmxdx∑n=1∞​an​∫−ππ​cosnxcosmxdx 项，只有当n≠mn \not = mn=m 时才不为 0，且∫−ππcosnxcosnxdx=π\int_{-π}^πcosnxcosnxdx = π∫−ππ​cosnxcosnxdx=π。所以：$$\int_-π}【πf_T(x)cosmxdx = a_nπ$$ $$a_n = {1\overπ】\int_{-π}^πf_T(x)cosmxdx, n = 1,2,3,…$$

求解 b_n 同理，对等式两边同时乘以 sinmx 后再从 -π 到 π 作定积分，可得：$$b_n = {1\overπ}\int_{-π}^πf_T (x) sinmxdx, n = 1,2,3,…$$

将 n = 0 代入 a_n 的通项公式中，发现a0=1π∫−ππfT(x)dx=2a0∗a_0 = {1 \over π} \int_{-π}^πf_T(x)dx = 2a_0^*a0​=π1​∫−ππ​fT​(x)dx=2a0∗​。为了保持一致性，改用 a_0 表达，即可得到周期为 2π 函数的傅里叶级数展开形式：$$f_T (x) = {a_0\over2} + \sum_{n = 1}^\infty (a_ncosnx + b_nsinnx)$$
