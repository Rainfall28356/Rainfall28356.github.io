1 三角函数的正交性

在解析几何中，定义两个向量正交可用点积（内积）来表示：\[\vec a \cdot \vec b = 0\]

将正交的概念推广到连续函数上，在区间\(x∈[a,b]\)，如果函数f(x)和g(x)正交，则\[\int_a^bf(x)g(x)dx = 0\]

此时定义三角函数系（省略0）：\[{1(cos0x),sinx,cosx,sin2x,cos2x,...,sinnx,cosnx,...}\]

可以知道任何两个不同函数在\(x∈[-π,π]\)内正交，即：\[\int_{-π}^πsinnxcosnxdx = 0\] \[\int_{-π}^πcosnxcosmxdx = 0(n\not=m)\] \[\int_{-π}^πsinnxdx = \int_{-π}^π1×sinnxdx = 0\]

注意只有当两个三角函数不相同时才可在\(x∈[-π,π]\)内正交，当\(n = m\)时（积化和差）： \[\begin{aligned} \int_{-π}^πcosnxcosnxdx &= \int_{-π}^π{1 \over 2}(1+cos2nx)dx \\ &= {1 \over 2}(\int_{-π}^π1dx + \int_{-π}^πcos2nxdx) \\ &= {1 \over 2}\int_{-π}^π1dx \\ &= π \end{aligned}\]

2 周期为2π的函数展开

对于一个周期\(T = 2π\)的函数：\[f_T(x) = f_T(x + 2π)\]

它总能用三角级数表达：\[f_T(x) = \sum_{n = 0}^\infty a_ncosnx + \sum_{n = 0}^\infty b_nsinnx\]

首先求解特殊项\(a_0^*\)。当\(n = 0\)时： \[\begin{aligned} f_T(x) &= a_0^*cos0 + b_0sin0 + \sum_{n = 1}^\infty (a_ncosnx + b_nsinnx) \\ &= a_0^* + \sum_{n = 1}^\infty (a_ncosnx + b_nsinnx) \end{aligned}\]

对等式两边从\(-π\)到\(π\)作定积分，得到： \[\begin{aligned} \int_{-π}^πf_T(x)dx &= \int_{-π}^πa_0^*dx + \int_{-π}^π\sum_{n = 1}^\infty (a_ncosnx + b_nsinnx)dx \\ &= \int_{-π}^πa_0^*dx + \sum_{n = 1}^\infty a_n \int_{-π}^πcosnxdx + \sum_{n = 1}^\infty b_n \int_{-π}^πsinnxdx \end{aligned}\]

根据三角函数的正交性，可以知道\(\sum_{n = 1}^\infty a_n \int_{-π}^πcosnxdx = \sum_{n = 1}^\infty b_n \int_{-π}^πsinnxdx = 0\)，因此：\[\int_{-π}^πf_T(x)dx = \int_{-π}^πa_0dx\] \[a_0^* = {1 \over 2π} \int_{-π}^πf_T(x)dx\]

下面求解通项\(a_n\)。对等式两边同时乘以\(cosmx\)后再从\(-π\)到\(π\)作定积分，得到：\[\int_{-π}^πf_T(x)cosmxdx = a_0^*\int_{-π}^πcosmxdx + \sum_{n = 1}^\infty a_n \int_{-π}^πcosnxcosmxdx + \sum_{n = 1}^\infty b_n \int_{-π}^πsinnxcosmxdx\]

对于\(\sum_{n = 1}^\infty a_n \int_{-π}^πcosnxcosmxdx\)项，只有当\(n \not = m\)时才不为0，且\(\int_{-π}^πcosnxcosnxdx = π\)。所以：\[\int_{-π}^πf_T(x)cosmxdx = a_nπ\] \[a_n = {1\overπ}\int_{-π}^πf_T(x)cosmxdx, n = 1,2,3,...\]

求解\(b_n\)同理，对等式两边同时乘以\(sinmx\)后再从\(-π\)到\(π\)作定积分，可得：\[b_n = {1\overπ}\int_{-π}^πf_T(x)sinmxdx, n = 1,2,3,...\]

将\(n = 0\)代入\(a_n\)的通项公式中，发现\(a_0 = {1 \over π} \int_{-π}^πf_T(x)dx = 2a_0^*\)。为了保持一致性，改用\(a_0\)表达，即可得到周期为\(2π\)函数的傅里叶级数展开形式：\[f_T(x) = {a_0\over2} + \sum_{n = 1}^\infty (a_ncosnx + b_nsinnx)\] \[ \left\{ \begin{aligned} a_n &= {1\overπ}\int_{-π}^πf_T(x)cosmxdx, n = 0,1,2,3,...\\ b_n &= {1\overπ}\int_{-π}^πf_T(x)sinmxdx, n = 1,2,3,... \end{aligned} \right.\]

3 周期为2L的函数展开

更一般地，对于一个周期\(T = 2L\)的函数：\[f_T(x) = f_T(x + 2L)\]

可以通过换元法，令\(t = {L\overπ}x\)，构造出周期\(T = 2π\)的函数\(g_T(x)\)：\[g_T(x + 2π) = f_T({L\overπ} (x + π) ) = f_T({L\overπ}x) = g_T(x)\]

将\(g_T(x)\)展开为傅里叶级数，得到：\[g_T(x) = {a_0\over2} + \sum_{n = 1}^\infty (a_ncosnx + b_nsinnx)\] \[ \left\{ \begin{aligned} a_n &= {1\overπ}\int_{-π}^πg_T(x)cosmxdx, n = 0,1,2,3,...\\ b_n &= {1\overπ}\int_{-π}^πg_T(x)sinmxdx, n = 1,2,3,... \end{aligned} \right.\]

将\(x = {π \over L}t\)代入，同时更换积分上下限为从\(-L\)到\(L\)，得到\(f_T(x)\)的傅里叶级数展开：\[f_T(t) = g_T(x) = {a_0 \over 2} + \sum_{n = 1}^\infty (a_ncosn{π \over L}t + b_nsinn{π \over L}t)\] \[ \left\{ \begin{aligned} a_n &= {1 \over π}\int_{-L}^Lf_T(t)cosn{π \ overL}td{π \over L}t = {1 \over L}\int_{-L}^Lf_T(t)cosn{π \over L}tdt, n = 0,1,2,3,...\\ b_n &= {1 \over π}\int_{-L}^Lf_T(t)sinn{π \over L}td{π \over L}t = {1 \over L}\int_{-L}^Lf_T(t)sinn{π \over L}tdt, n = 1,2,3,... \end{aligned} \right.\] 定义一个新变量基频率\(\omega\)，令\[\omega = {\pi \over L} = {2\pi \over T}\]

可以得到一般周期函数的傅里叶级数展开形式：\[f_T(t) = {a_0 \over 2} + \sum_{n = 1}^\infty (a_ncosn\omega t + b_nsinn\omega t)\] \[ \left\{ \begin{aligned} a_n &= {2 \over T}\int_{-{T \over 2} }^{T \over 2}f_T(t)cosn\omega tdt, n = 0,1,2,3,...\\ b_n &= {2 \over T}\int_{-{T \over 2} }^{T \over 2}f_T(t)sinn\omega tdt, n = 1,2,3,... \end{aligned} \right.\]

如果\(f_T(x)\)的周期T无限大，此时可将其看作非周期函数\(f(t)\)。但在展开具体研究之前，需要找到傅里叶级数的复数表达形式。

4 复数表达形式

欧拉恒等式将三角函数与复数联系到了一起，可以知道：\[cos\phi + isin\phi = e^{i\phi}\] \[cos\phi = {1 \over 2}(e^{i\phi} + e^{-i\phi})\] \[sin\phi = -{1 \over 2}i(e^{i\phi} - e^{-i\phi})\]

将其代入周期函数的傅里叶级数展开形式，得到： \[\begin{aligned} f_T(t) &= {a_0 \over 2} + \sum_{n = 1}^\infty (a_ncosn\omega t + b_nsinn\omega t) \\ &= {a_0 \over 2} + \sum_{n = 1}^\infty ({a_n \over 2}(e^{in\omega t} + e^{-in\omega t}) - {ib_n \over 2}(e^{in\omega t} - e^{-in\omega t})) \\ &= {a_0 \over 2} + \sum_{n = 1}^\infty ({ {a_n - ib_n} \over 2}e^{in\omega t} + { {a_n + ib_n} \over 2}e^{-in\omega t}) \\ &= {a_0 \over 2} + \sum_{n = 1}^\infty { {a_n - ib_n} \over 2}e^{in\omega t} + \sum_{n = 1}^\infty{ {a_n + ib_n} \over 2}e^{-in\omega t} \end{aligned}\]

将右边第1、3项作以下变形：\[{a_0 \over 2} = \sum_{n = 0}^0{a_0 \over 2}e^{in\omega t}\] \[\sum_{n = 1}^\infty{ {a_n + ib_n} \over 2}e^{-in\omega t} = \sum_{n = -\infty}^{-1}{ {a_{-n} + ib_{-n}} \over 2}e^{in\omega t}\]

发现加和中的n覆盖了\([-\infty, \infty]\)中的全部整数，因此可以整合为统一形式：\[f_T(t) = \sum_{n = {-\infty}}^\infty c_ne^{in\omega t}\] \[c_n = \begin{cases} {a_0 \over 2},\qquad \quad n = 0 \\ { {a_n - ib_n} \over 2}, \quad \; \ n = 1,2,3,... \\ {a_{-n} + ib_{-n} \over 2}, \ \ \ n = -1,-2,-3,... \\ \end{cases} \]

接着将\(a_n,b_n\)通项代入其中，当\(n = 0\)时： \[\begin{aligned} c_n &= {a_0 \over 2} \\ &= {1 \over T}\int_{-{T \over 2} }^{T \over 2}f_T(t)e^{-i0\omega t}dt \end{aligned}\]

当\(n = 1,2,3,...\)时： \[\begin{aligned} c_n &= { {a_n - ib_n} \over 2} \\ &= {1 \over T}\int_{-{T \over 2} }^{T \over 2}f_T(t)(cosn\omega t - isinn\omega t)dt \\ &= {1 \over T}\int_{-{T \over 2} }^{T \over 2}f_T(t)e^{-in\omega t}dt \end{aligned}\]

当\(n = -1,-2,-3,...\)时： \[\begin{aligned} c_n &= {a_{-n} + ib_{-n} \over 2} \\ &= {1 \over T}\int_{-{T \over 2} }^{T \over 2}f_T(t)(cos(-n)\omega t + isin(-n)\omega t)dt \\ &= {1 \over T}\int_{-{T \over 2} }^{T \over 2}f_T(t)e^{-in\omega t}dt \end{aligned}\]

至此，便可将所有情况统一为一种形式，即：\[f_T(t) = \sum_{n = {-\infty}}^\infty c_ne^{in\omega t}\] \[c_n = {1 \over T}\int_{-{T \over 2} }^{T \over 2}f_T(t)e^{-in\omega t}dt, n = 0, \pm 1, \pm 2, \pm 3,...\]

5 傅里叶变换

观察\(f_T(t)\)的傅里叶级数展开式，可以发现将区别不同函数的关键是\(c_n\)项，而每一个频率\(\omega\)都对应着一个\(c_n\)值。

当周期\(T\rightarrow \infty\)时，两个频率间的间隔\(\Delta \omega = \omega \rightarrow 0\)，此时频域的横坐标变成了连续变量\(\omega\)，频域函数也将成为一条连续曲线。

将\(c_n\)代入傅里叶级数的复数表达式： \[\begin{aligned} f_T(t) &= \sum_{n = {-\infty}}^\infty {1 \over T}\int_{-{T \over 2} }^{T \over 2}f_T(t)e^{-in\omega t}dte^{in\omega t} \\ &= {\omega \over {2\pi } }\sum_{n = {-\infty}}^\infty \int_{-{T \over 2} }^{T \over 2}f_T(t)e^{-in\omega t}dte^{in\omega t} \\ &= {1 \over {2\pi } }\sum_{n = {-\infty}}^\infty \int_{-{T \over 2} }^{T \over 2}f_T(t)e^{-in\omega t}dte^{in\omega t} \Delta \omega \end{aligned}\]

当\(T\rightarrow \infty\)时，\(f_T(t)\rightarrow f(t)\)，\(\int_{-{T \over 2} }^{T \over 2}f_T(t)e^{-in\omega t} \rightarrow \int_{-{\infty} }^{\infty}f_T(t)e^{-in\omega t}\)，连加变为积分，得到：\[f(t) = {1 \over {2\pi } }\int_{-{\infty} }^{\infty}\int_{-{\infty} }^{\infty}f_T(t)e^{-in\omega t}dte^{i\omega t}d\omega\]

其中，定义 \[F(\omega) = \int_{-{\infty} }^{\infty}f_T(t)e^{-in\omega t}dt\] \(F(\omega)\)就被称为\(f(t)\)的傅里叶变换，而\(f(t) = {1 \over {2\pi } }\int_{-{\infty} }^{\infty}F(\omega)e^{i\omega t}d\omega\) 则是 \(F(\omega)\) 的傅里叶逆变换。

特别的，当\(s = i\omega\)时就是我们所说的拉普拉斯变换：\[F(s) = \int_0^{\infty}f(t)e^{-st}dt\]
